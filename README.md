# Diet Plan Generator

A full-stack web application that generates personalized diet plans based on user inputs such as weight, height, age, gender, activity level, and fitness goals. The app provides a meal plan for breakfast, lunch, dinner, and snacks, along with the ability to download the plan as a PDF.

---

## Features

- User inputs: Name, Age, Weight, Height, Gender, Activity Level, Goal, Diet Preference
- Calculates current calorie intake and target calories based on user goals
- Generates a personalized meal plan (Breakfast, Lunch, Dinner, Snacks) with portion sizes
- Downloadable meal plan as a PDF
- Responsive and modern frontend UI
- Fully functional backend API with FastAPI

---
